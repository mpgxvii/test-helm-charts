Thank you for deploying platform-backend.

To install with defaults:

  helm install {{ .Release.Name }} {{ .Chart.Name }} -n <namespace>

To override image tags and enable ingress for delegate-api:

  helm install {{ .Release.Name }} {{ .Chart.Name }} \
    --set services.delegate-api.image.tag=1.0.0 \
    --set services.delegate-api.ingress.enabled=true \
    --set services.delegate-api.ingress.hosts[0].host=delegate.example.com \
    -n <namespace>

To verify resources:

  kubectl get deploy,svc,ingress,hpa -l app.kubernetes.io/instance={{ .Release.Name }}

Each service can be enabled/disabled and configured under .Values.services.<name>.
